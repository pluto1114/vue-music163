# vue-music163

> A Vue.js project for music
### Use
* vuejs-2.0
* vue-cli
* vue-router
* vuex
* mint-ui


## Support
* Chrome
* Firefox
* Mobile browser

## Package for app
* Appcan
* Apicloud

## Other link
- vue-music-qq : https://github.com/pluto1114/vue-music-qq

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
